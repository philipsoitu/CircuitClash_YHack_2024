SHDL (Simple Hardware Description Language) Documentation

SHDL is a language designed for educational purposes, focusing on demystifying digital logic and circuit design principles through programming.

Language Structure

An SHDL program consists of the following components:

  • Variable Declarations: Declare the inputs, outputs, and internal wires used in the circuit.
  • Gate Definitions: Define custom gates that encapsulate specific logical functionalities.
  • Gate Instantiations: Instantiate gates and define connections between them to form a digital circuit.

Variables

Variables in SHDL play essential roles in defining the circuit's behavior:

  • Inputs: Represent external signals entering the circuit.
  • Outputs: Represent signals leaving the circuit.
  • Wires: Internal connections facilitating communication between gates.
        INPUT A
        OUTPUT B
        WIRE C

Gates

Gates are fundamental building blocks in SHDL, encapsulating specific logical operations:

  • Gate Definition: Custom gates are defined using the STARTGATE and ENDGATE keywords, specifying their inputs and outputs.
  • Constraints: Gates cannot be nested within each other or invoke themselves recursively to prevent infinite loops.
        STARTGATE GATE1 INPUT A OUTPUT B
            // Gate definition here
        ENDGATE

Syntax Highlights

Key syntax elements in SHDL include:

  • Variable Declaration: Declare variables using INPUT, OUTPUT, or WIRE keywords.
  • Gate Definition: Define gates using the STARTGATE and ENDGATE keywords.
  • Operations: Perform logical operations such as AND, OR, NOT, and XOR.
        INPUT <variable_name>
        OUTPUT <variable_name>
        WIRE <variable_name>

        STARTGATE <gate_name> INPUT <input1> [INPUT <input2> ...] OUTPUT <output1> [OUTPUT <output2> ...]
            // Operations here
        ENDGATE

        AND, OR, NOT, XOR

Example

Below is an example SHDL code illustrating the creation of a digital circuit:

        INPUT A
        INPUT B
        OUTPUT S
        OUTPUT C
        WIRE T1
        WIRE T2

        STARTGATE MYXOR INPUT X INPUT Y OUTPUT Z
            OR X Y T1
            AND X Y T2
            NOT T2 T3
            AND T1 T3 Z
        ENDGATE

        STARTGATE ADDER INPUT X INPUT Y OUTPUT Z OUTPUT I    
            MYXOR X Y Z
            AND X Y I
        ENDGATE

        ADDER A B S C

Conclusion

SHDL simplifies the exploration of digital logic and circuit design principles, making it an invaluable tool for educational contexts.
